# Idea Inbox Repository

## この repository について

この repository は、  
**まだ形になっていないアイデア・違和感・調査ネタを一時的に受け止めるための Inbox** です。

ここは開発を行う場所ではありません。  
どのサービス・どのリポジトリに属するかが決まっていない思考を、安心して置いておくための場所です。

---

## ここに書いてよいもの

- ひと言アイデア
- 違和感やモヤっとした気づき
- サービス案・運用案・調査案
- 技術／非技術が混ざった未整理メモ

完成度は問いません。  
**雑なままで問題ありません。**

---

## ここでやらないこと

- 要件定義
- 仕様整理
- 影響範囲の検討
- 優先度付け
- 担当者割り当て

ここは「考えを育てる前の置き場」です。  
整理や判断は、後段で行います。

---

## Inbox Issue の扱い方

- Inbox Issue は原則 1つだけ作成します
- コメント 1つ = アイデア 1つ
- 背景説明や理由は不要です
- Issue は Close しません

処理済みかどうかは、コメント単位で判断します。

---

## Issue 化について

この Inbox に溜まった内容は、AI エージェントを用いて、必要に応じて正式な Issue に変換します。

- どの repository に属するか
- 機能改修か、調査か、改善か

といった判断は、**後から行います**。

人が無理に整理せず、AI と分業することを前提とした運用を想定しています。
